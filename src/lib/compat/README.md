# Zig Compatibility Libraries

The following directory contains code that either used to exist in the Zig
standard library, or currently exists in Zig master that has been deemed
useful to have ahead of a full release.

Code located here should likely be eventually phased out, or moved to a more
logical location if it's decided it would be useful to keep.
